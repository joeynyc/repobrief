How It Works
Three commands. That's it.
No config files. No setup. No API keys.
1

Initialize

Run npx repobrief init in any git repo. RepoBrief scans your code, dependencies, git history, and project structure in seconds.

2

Export

Run repobrief export --format claude to generate optimized context for your AI tool of choice. Supports Claude, Cursor, Codex, and markdown.

3

Stay current

Run repobrief update after major changes. Your context evolves with your code ‚Äî not a stale doc that was accurate three months ago.
